Web API

The web API endpoints return data about the phones by using REST principles. This data comes from the database and is described in JSON form.

HTTPS Requests

The Web API uses REST principles. This means that the Data is accessed through HTTPS requests that have an specific endpoint. The method of the request depends on its action.

Method Action
POST Create a new resource
GET Retrieve a resource
PUT Update a resource
DELETE Delete a resource

Website Resources

These are the requests and responses of the API. The recourses and data is transmitted with JSON.
